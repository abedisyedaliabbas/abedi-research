# Travel Photos Directory

## How to Add Your Photos

1. **Copy your photos** from your website or computer
2. **Rename them** to match the expected file names (see list below)
3. **Place them** in this directory (`assets/img/travel/`)

## Supported Formats
- `.jpg` or `.jpeg` (recommended)
- `.png`
- `.webp`

## Tips
- Keep file sizes reasonable (under 2MB per image for faster loading)
- Use descriptive file names
- Make sure photos match the correct locations!
