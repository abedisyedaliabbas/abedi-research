# Main Page Photos Directory

## How to Add Your Photos

1. **Copy your photos** from your website or computer
2. **Rename them** to match the file names below
3. **Place them** in this directory (`assets/img/main/`)

## Required Photo Files (5 photos):

### Hero Section:
1. **`profile-photo.jpg`** (Professional headshot)
   - Used in: Hero section on main page
   - Display: Circular profile photo (280x280px)
   - Should be a professional headshot/portrait photo
   - If missing, a placeholder icon will be shown

### University Logos for Affiliations Section:

1. **`sutd.jpg`**
   - University: Singapore University of Technology and Design (SUTD)
   - Used in: Affiliations section on main page
   - Display: University logo/photo

2. **`ntu.jpg`**
   - University: Nanyang Technological University (NTU)
   - Used in: Affiliations section on main page
   - Display: University logo/photo

3. **`ucl.jpg`**
   - University: University College London (UCL)
   - Used in: Affiliations section on main page
   - Display: University logo/photo

4. **`queen-mary.jpg`**
   - University: Queen Mary University of London
   - Used in: Affiliations section on main page
   - Display: University logo/photo

## Supported Formats
- `.jpg` or `.jpeg` (recommended)
- `.png`
- `.webp`

## Tips
- **Profile photo**: Should be a professional headshot, square format works best
- **University logos**: Photos should be university logos or photos related to the universities
- Recommended size: 200-300px width (logos will be automatically resized)
- Keep file sizes reasonable (under 1MB per image for faster loading)
- If a photo is missing, a fallback icon will be displayed

## Where They Appear

- **Profile photo**: Hero section on the main page (right side, circular)
- **University logos**: Affiliations section on the main page, showing your connections with these universities
